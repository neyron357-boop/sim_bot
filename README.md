# SIM Bot

Проект теперь поддерживает 2 интерфейса, работающих с одной и той же базой данных `sim_bot.db`:

- `bot.py` — Telegram-бот.
- `web_app.py` — веб-панель на Flask.

## Запуск

```bash
pip install -r requirements.txt
python bot.py
```

Отдельно (или на другом процессе) можно запустить веб-панель:

```bash
python web_app.py
```

По умолчанию веб доступен на `http://localhost:8080`.

## Синхронизация с ботом

Синхронизация реализована через общую SQLite БД (`sim_bot.db`):

- Любые изменения из веб-панели сразу видны боту.
- Любые изменения из бота сразу видны в веб-панели после обновления страницы.

При необходимости можно указать другую БД через переменную окружения:

```bash
export SIM_BOT_DB=/path/to/sim_bot.db
python web_app.py
```
