# SIM Bot

Telegram-бот для учета SIM-тарифов сотрудников.

## Что умеет

- добавление, удаление и редактирование записей сотрудников;
- хранение данных в SQLite (`sim_bot.db`) с миграцией из `sim_users.json`;
- кошелек с точными денежными расчётами (в центах), журналом операций и аудитом;
- сохранение тарифов с ценой **и длительностью в днях**;
- ежедневные уведомления по срокам: за 7, 3, 1 день, в день окончания и по просроченным;
- контроль админ-доступа через `ADMIN_CHAT_IDS` или автоназначение первого администратора.

## Запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Укажите переменные окружения:
   ```bash
   export BOT_TOKEN=<telegram_token>
   # опционально, список chat_id админов через запятую
   export ADMIN_CHAT_IDS=123456789,987654321
   ```
3. Запустите бота:
   ```bash
   python bot.py
   ```

## Примечания по данным

- При первом запуске бот создаёт `sim_bot.db`.
- Если найден старый `sim_users.json`, бот автоматически переносит данные в SQLite.
