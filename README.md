# SIM Bot

Telegram-бот для учета SIM-тарифов сотрудников:

- добавление, удаление и редактирование записей;
- хранение данных в `sim_users.json`;
- учет кошелька и стоимости тарифов;
- ежедневные уведомления по сроку окончания тарифа (за 3 дня и за 1 день).

## Запуск

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Создайте `.env` на основе примера и укажите токен:
   ```bash
   cp .env.example .env
   # затем отредактируйте .env и задайте BOT_TOKEN
   ```
3. Экспортируйте переменные окружения и запустите бота:
   ```bash
   export $(grep -v '^#' .env | xargs)
   python bot.py
   ```
